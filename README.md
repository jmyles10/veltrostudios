🚀 How to Update the Site

Option A — GitHub Website (quick edits)


Go to the repo on github.com
Click index.html → click the ✏️ pencil icon (top right)
Make your changes
Scroll down → click "Commit changes"
Site updates in ~60 seconds


Option B — GitHub Desktop (recommended)


Open GitHub Desktop
Click "Fetch origin" to pull the latest version first
Replace or edit index.html on your computer
GitHub Desktop will detect the change automatically
Add a commit message describing what you changed
Click "Commit to main" → then "Push origin"
Site updates in ~60 seconds



⚠️ Always fetch/pull before editing to avoid overwriting each other's work.
